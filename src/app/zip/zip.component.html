zip

Combines values from both observables, emitting a pair of values only when both observables emit. ⚠️ Watch every 3 secs.

Use Case: When you have two observables that emit values at different times but need to combine their emissions in a one-to-one manner, i.e., first emitted value from each observable goes together, second emitted value goes together, and so on.
Example: Fetching a user's profile and their comments, and you want to pair each profile with each comment at the same index.

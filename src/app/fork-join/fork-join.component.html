forkjoin

Waits for all source observables to complete. Emits the last emitted value from each observable as an array when all are done.

Use Case: When you want to wait for all observables to complete and then process the final values together. All must emit and complete.
Example: You need to fetch a user, their posts, and their comments, but you want to process everything once all are finished.

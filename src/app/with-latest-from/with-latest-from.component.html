with latest from

Pairs the latest value from another observable (triggerObservable 10 secs) with each emission of the main observable (dataObservable 5 secs). ⚠️ Watch every 5 secs.

Use Case: When you have a “trigger” observable (like a button click), and you want to combine it with the latest value from another observable (but only when the trigger emits).
Example: Clicking a button to submit a form and you want to get the latest form values at that moment.

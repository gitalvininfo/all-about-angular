combine latest

Combines values from both observables, emitting a pair of values only when both observables emit. ⚠️ Watch every 5 secs.

Use Case: When you want to combine the latest value from multiple observables, regardless of when they emit. Each time any observable emits, you get the latest value from all the others. Example: Imagine getting user data from one API and post data from another, and you always want the latest user and the latest post combined together.
